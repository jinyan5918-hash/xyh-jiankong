企业版抖音点赞监控 — Windows 客户端

使用步骤
1. 解压本文件夹到任意位置（勿删除文件夹内其他文件）。
2. 双击文件夹内的 exe（名称形如 EnterpriseDouyinClient-1.2.0.exe，版本号随发行变化）。
3. 窗口标题应显示「企业版抖音点赞监控客户端 v版本号」，便于确认是否为最新包。
4. 使用管理员分配的用户名、密码登录。

说明
- 请勿单独移动或删除与 exe 同目录的 config.json。
- 请阅读同目录下的「更新日志.txt」了解本版更新内容与用法。
- 同目录下的 build_info.txt（若存在）含本次打包对应的 Git 提交号，可与管理员核对是否为最新构建。
- 若无法登录，请检查网络或联系管理员确认服务器地址与账号。
