【使用说明】
--------------------------------------------------------------------------------
1. 将整个文件夹解压到任意目录，勿删除同目录下的 config.json、release_version.txt、user_prefs.json（若有）等文件。
2. 双击「EnterpriseDouyinClient-版本号.exe」启动；在欢迎页登录。账号与密码由管理员在后台创建/分配。
3. config.json 中的 api_base 为服务器地址（与浏览器访问后台时一致，不要带 /admin）。
4. 员工填写 Webhook 后务必点「保存企业微信通知」；管理员也可在后台预填。测试企微前请确认服务器已部署最新 API 并已重启。
5. 若无法登录或收不到企微消息，请把本文件中的「版本号」「更新日期」及 build_info.txt（若有）发给管理员排查。
